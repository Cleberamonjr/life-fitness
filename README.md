# Life Fitness

App pessoal de acompanhamento de treino, dieta, hidratação e composição corporal.

## Funcionalidades

- Diário matinal — hora que acordou, peso, cintura
- Tracker de hidratação (meta 3.5L/dia)
- Checklist de suplementos
- Plano alimentar com macros em tempo real
- Treino interativo com 4 dias (A/B/C/D), bisets e registro de carga
- Estimativa de calorias queimadas com base na carga utilizada
- Histórico de cargas por exercício (auto-preenchimento)
- Estimativa de % de gordura pelo Método Naval
- Silhuetas de referência em 7%, 10%, 12%, 15%, 18% e 20%
- Gráficos de peso, cintura e quadril ao longo do tempo
- Dashboard com anéis de progresso, barras de água e saldo calórico
- Streak de dias completos

## Como usar

Acesse via GitHub Pages após configurar (veja abaixo).

## Deploy via GitHub Pages

1. Crie um repositório chamado `life-fitness` no GitHub
2. Faça upload do arquivo `index.html`
3. Vá em **Settings → Pages**
4. Em **Source**, selecione **Deploy from a branch**
5. Branch: `main`, pasta: `/ (root)`
6. Clique em **Save**
7. Aguarde ~2 minutos e acesse: `https://SEU_USUARIO.github.io/life-fitness`

## Tecnologias

- HTML5 / CSS3 / JavaScript puro
- Chart.js (CDN)
- localStorage para persistência de dados
